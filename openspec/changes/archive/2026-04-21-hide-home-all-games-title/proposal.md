## Why

The home page currently shows a section label reading “全部游戏”, which adds visual noise without adding useful information. Since the page already presents the available games as cards, removing the label makes the entry screen cleaner and more focused.

## What Changes

- Remove the “全部游戏” section label from the home page game list.
- Keep the existing game cards, navigation behavior, and overall page structure unchanged.
- Preserve the current home page styling and spacing as much as possible while removing the unused label.

## Capabilities

### New Capabilities
- `home-page`: Defines how the home page presents the game entry list and related page content.
