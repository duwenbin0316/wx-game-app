## ADDED Requirements

### Requirement: Home page game list has no section label
The home page SHALL display the available game cards without showing the “全部游戏” section label above the list.

#### Scenario: Home page renders game cards
- **WHEN** a user opens the home page
- **THEN** the page shows the existing game cards
- **AND** the page does not show the “全部游戏” label above the list
